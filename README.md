# electricity-api ⚡️...🌍

###Start project
```bash
make build
```

###Run test
```bash
make test-unit
```

###Existing commands
```bash
make electricity:search-suspicious-readings <filename>
```
Note: `<filename>` value is required for this command. For example, it could be `2016-readings.xml` or `2016-readings.csv`
